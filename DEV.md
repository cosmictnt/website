# DEV

You need:
- git
- npm
- vscode

1. Get your code:

`git clone https://github.com/cosmictnt/cosmictnt.github.io.git D:\website`

2. switch to dev

`git branch -B dev`
`git branch --set-upstream-to=origin/dev dev`
`git pull`

3. make your changes

npm install
npm run dev

4. commit and push using the vscode git system


# Updating your website

5. Make a pull request online
6. merge dev with main
7. DONT DELETE DEV BRANCH
